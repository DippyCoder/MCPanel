Create Server

Spigot requires manual compilation via BuildTools. The server folder will be created but no JAR will be downloaded. Place your compiled spigot.jar in the server folder after creation.

Create Profile

After creating the profile, click "Open Folder" to add your plugins, configs, and other files. Everything inside the profile folder (except profile.json) will be copied to the server on creation.

Import Profile

Files from the selected folder will be copied into a new profile. If the folder contains a profile.json, its name and description will be pre-filled automatically.

Import Server

The selected folder's contents will be copied into MCPanel's servers directory. Port and software type are auto-detected from server.properties when you browse.
